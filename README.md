 <b data-path-to-node="0" data-index-in-node="31">1.2.3 update</b> focuses on <b data-path-to-node="0" data-index-in-node="55">operational automation</b>, <b data-path-to-node="0" data-index-in-node="79">real-time previews</b>, and <b data-path-to-node="0" data-index-in-node="103">task management flexibility</b>.</p>
  <p data-path-to-node="1">key updates:</p>
<h3 data-path-to-node="3">1. Task Selection and Workflow Automation</h3>
  <ul data-path-to-node="4"><li><p data-path-to-node="4,0,0"><b data-path-to-node="4,0,0" data-index-in-node="0">From "Mark Completed" to "Toggle Enable":</b> The new version replaces the simple completion icon with a functional checkbox; only checked tasks will be included in the batch.</p></li>
    <li><p data-path-to-node="4,1,0"><b data-path-to-node="4,1,0" data-index-in-node="0">Auto-Deselect upon Completion:</b> Once a task finishes rendering, the system automatically unchecks it to prevent redundant rendering in the next session.</p></li><li><p data-path-to-node="4,2,0"><b data-path-to-node="4,2,0" data-index-in-node="0">Bulk Toggle Buttons:</b> New "Select All" and "Deselect All" buttons have been added to the bottom for easier management of large task lists.</p></li></ul>
  <h3 data-path-to-node="5">2. Real-time Preview and Switching</h3><ul data-path-to-node="6">
    <li><p data-path-to-node="6,0,0"><b data-path-to-node="6,0,0" data-index-in-node="0">Click Name to Preview:</b> When a task is collapsed, clicking its name automatically switches the 3D viewport to that specific <b data-path-to-node="6,0,0" data-index-in-node="123">View Layer</b> and <b data-path-to-node="6,0,0" data-index-in-node="138">Camera</b>.</p></li>
    <li><p data-path-to-node="6,1,0"><b data-path-to-node="6,1,0" data-index-in-node="0">Instant Property Sync:</b> Modifying the View Layer or Camera directly within the panel now triggers an immediate update in the active scene, providing a "what you see is what you get" experience.</p></li></ul><h3 data-path-to-node="7">3. UI and Visual Enhancements</h3><ul data-path-to-node="8">
    <li><p data-path-to-node="8,0,0"><b data-path-to-node="8,0,0" data-index-in-node="0">Visual Status Distinction:</b> Tasks that are unchecked (disabled) now appear <b data-path-to-node="8,0,0" data-index-in-node="74">greyed out</b> in the interface, making it easy to distinguish which tasks will be skipped.</p></li><li><p data-path-to-node="8,1,0"><b data-path-to-node="8,1,0" data-index-in-node="0">Clean Layout:</b> The interface has been streamlined by removing redundant icons and organizing input fields more tidily when a task is expanded.</p></li></ul><h3 data-path-to-node="9">4. System Stability and Logic</h3><ul data-path-to-node="10">
    <li><p data-path-to-node="10,0,0"><b data-path-to-node="10,0,0" data-index-in-node="0">Smart Rename Tracking:</b> A new message bus system tracks View Layer name changes to prevent broken links within the addon.</p></li>
    <li><p data-path-to-node="10,1,0"><b data-path-to-node="10,1,0" data-index-in-node="0">Smarter Defaults:</b> New tasks now automatically detect and fill in the scene's existing View Layers and Cameras, reducing the need for manual setup.</p></li></ul><hr data-path-to-node="11"></div>

  <p data-path-to-node="1">以下是具體的更新重點：</p>
  <h3 data-path-to-node="3">1. 任務選取與自動化流程</h3><ul data-path-to-node="1"><li>
    <p data-path-to-node="4,0,0"><b data-path-to-node="4,0,0" data-index-in-node="0">從「標記完成」改為「勾選啟動」：</b> 新版本不再只是顯示一個打勾符號，而是新增了實質的勾選框（Checkbox）。只有被勾選的任務才會進入排程。</p>
  </li><li><p data-path-to-node="4,1,0"><b data-path-to-node="4,1,0" data-index-in-node="0">自動取消勾選：</b> 當某個任務算圖完成後，系統會<b data-path-to-node="4,1,0" data-index-in-node="22">自動取消</b>該任務的勾選。這樣下次按下算圖時，就不會重複算到已經完成的東西。</p>
  </li><li><p data-path-to-node="4,2,0"><b data-path-to-node="4,2,0" data-index-in-node="0">全選與全不選：</b> 底部新增了「Select All」和「Deselect All」按鈕，方便快速管理大量的渲染任務。</p>
  </li></ul><h3 data-path-to-node="5">2. 即時預覽與切換功能</h3><ul data-path-to-node="6"><li>
    <p data-path-to-node="6,0,0"><b data-path-to-node="6,0,0" data-index-in-node="0">點擊名稱切換場景：</b> 當任務縮小收合時，點擊任務名稱會自動幫你切換到該任務設定的<b data-path-to-node="6,0,0" data-index-in-node="39">視圖層 (View Layer)</b> 與<b data-path-to-node="6,0,0" data-index-in-node="57">攝影機 (Camera)</b>，讓你不用手動調整就能快速檢查設定。</p></li><li><p data-path-to-node="6,1,0"><b data-path-to-node="6,1,0" data-index-in-node="0">即時同步：</b> 在面板裡修改 View Layer 或 Camera 時，3D 視圖會立即跟著連動切換，達到「所見即所得」的效果。</p></li></ul><h3 data-path-to-node="7">3. UI 介面優化</h3>
  <ul data-path-to-node="8"><li><p data-path-to-node="8,0,0"><b data-path-to-node="8,0,0" data-index-in-node="0">視覺狀態區分：</b> 沒有勾選（不渲染）的任務，在介面上會顯示為<b data-path-to-node="8,0,0" data-index-in-node="29">灰色半透明狀態</b>，讓你能直觀地分辨哪些任務是略過的。</p>
  </li><li><p data-path-to-node="8,1,0"><b data-path-to-node="8,1,0" data-index-in-node="0">介面簡化：</b> 移除了一些重複的圖示，讓排版更乾淨，並在任務展開時提供更整齊的參數輸入框。</p>
  </li></ul><h3 data-path-to-node="9">4. 系統穩定性</h3><ul data-path-to-node="10"><li><p data-path-to-node="10,0,0"><b data-path-to-node="10,0,0" data-index-in-node="0">智慧更名追蹤：</b>
